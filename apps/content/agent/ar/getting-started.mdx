## Quick Setup

### Accessing the Portal
1. Open your web browser
2. Navigate to: **https://crm.newyolk.io**
3. Bookmark this URL for easy access

### Initial Login
1. **Agent ID**: Enter the unique ID provided by your company
2. **Password**: Use the temporary password given to you
3. Click **Login**

> **First-time users**: You'll be automatically prompted to create a new secure password for future logins.
